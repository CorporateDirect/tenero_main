# Shopify Custom Meta Objects


## Episodes

- Meta Object type: "episode"

### Fields
Meta Field | Key
Episode Title | "episode_title"
Episode Number | "episode_number"
Episode Short Description | "episode_short_description"
Episode Thumbnail | "episode_thumbnail"
Episode Tags | "episode_tags"
Episode URL | "episode_url"
Publish Date | "publish_date"
Show | "show" (reference to the show metaobject used on the page)

## Show Episode Tags

- Meta Object type: "show_episode_tag"

### Fields
Meta Field | Key
Tag | "tag"
